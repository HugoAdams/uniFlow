Get the Service Account Key As JSON

The following process is based on Getting the Service Account key.
  1. Click on the gear icon, to the right of the agent name.
  2. Under the GOOGLE PROJECT section, click on the name of the Service Account.
  3. Click on the Create Service Account button at the top of the page.
  4. In the pop up, enter a name for the service account.
  5. Click on Role. Under the Dialogflow category, select Dialogflow API Admin.
  6. Check the Furnish a new private key option and make sure JSON is selected for Key type.
  7. Click the Create button.
